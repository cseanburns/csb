Notes

I used to use Octopress and then Dokuwiki to keep notes on various topics I study. I've developed different work flows in more receny years, but wanted to preserve these notes on this site.

Most of the notes were saved in DokuWiki syntax. I use pandoc to convert them to HTML. In some cases, such as with the notes on R, I found it better to convert my notes to Markdown and then to HTML.

I converted these notes from Markdown to HTML with the following command:


pandoc -f markdown -t html --mathml file.md -s -o file.html

The Library Quarterly

Notes on a reading of the first four volumes of The Library Quarterly

  1. Notes on The Library Quarterly

I converted these notes from DokuWiki syntax to HTML with the following command:

  
pandoc -f dokuwiki -t html5 \
  --metadata-file=metadata.yaml \
  --toc=true \
  background.txt literature-review.txt \
  volume-1-issue-1.txt volume-1-issue-2.txt volume-1-issue-3.txt volume-1-issue-4.txt \
  volume-2-issue-1.txt volume-2-issue-2.txt volume-2-issue-3.txt volume-2-issue-4.txt \
  volume-3-issue-1.txt volume-3-issue-2.txt volume-3-issue-3.txt volume-3-issue-4.txt \
  volume-4-issue-1.txt volume-4-issue-2.txt volume-4-issue-3.txt volume-4-issue-4.txt -s -o library-quarterly-notes.html

The metadata.yaml file contained the following:

  
---
title: 'The Scholarly Journal and Disciplinary Identity'
author: 'C. Sean Burns'
date: '2013-2014'
description: |
  These notes are based on my reading
  of the first four volumes of the journal
  The Library Quarterly.
abstract: |
  These notes are based on my reading
  of the first four volumes of the journal
  The Library Quarterly. I started them on
  July 19, 2013 and finished on January 16, 2014.
  I'll continue to revise these notes
  and edit out DokuWiki specific aspects.
---
